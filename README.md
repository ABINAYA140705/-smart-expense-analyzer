# 💰 Smart Expense Analyzer with AI Insights

A full-stack FinTech web application to track, categorize, and analyze 
personal expenses with AI-powered saving suggestions.

## 🚀 Features
- Add & manage daily expenses
- Upload bulk expenses via CSV
- Interactive dashboard with pie & bar charts
- AI-based spending insights & suggestions
- Monthly budget limits with alerts
- Export expense report as PDF
- Edit & delete expenses

## 🛠 Tech Stack
| Layer | Technology |
|---|---|
| Frontend | React + Tailwind CSS + Recharts |
| Backend | Python Flask + REST API |
| Database | SQLite |
| Analytics | Pandas |
| PDF Export | ReportLab |

## ⚙️ How to Run

### Backend
cd backend
pip install -r requirements.txt
python app.py

### Frontend
cd frontend
npm install
npm run dev

Open http://localhost:5173
